# My First Deployment 🚀

A simple calculator app to learn the complete software deployment workflow!

## What This Project Does

This is a learning project that demonstrates:
- Writing Python code
- Writing automated tests
- Using Git for version control
- Pushing code to GitHub
- Automated testing with GitHub Actions (CI/CD)

## The Calculator

Simple calculator with these functions:
- ➕ Add two numbers
- ➖ Subtract two numbers
- ✖️ Multiply two numbers
- ➗ Divide two numbers
- 👋 Greet users by name

## How to Run

```bash
# Run the calculator
python calculator.py

# Run the tests
pytest test_calculator.py -v
```

## Learning Journey

This project was created to understand:
1. **Phase 1**: Creating code and tests
2. **Phase 2**: Setting up Git and GitHub
3. **Phase 3**: Configuring automated testing (CI/CD)
4. **Phase 4**: Making changes and creating Pull Requests
5. **Phase 5**: Debugging failed tests
